# DigiTools Platform

DigiTools Platform is a premium digital tools landing page where users can explore curated products, add them to a cart, review pricing plans, and complete a simple checkout flow. The project is built as a responsive single-page React application with a Figma-inspired UI.

## Technologies Used

- React.js
- Vite
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify

## Features

- Users can switch between the product view and cart view without leaving the page, which makes the shopping flow feel fast and simple.
- The cart updates in real time, including the navbar count, remove actions, total price calculation, and checkout clear action.
- The interface includes multiple polished sections like banner, stats, pricing, steps, and footer, all designed to feel like one complete product landing page.
